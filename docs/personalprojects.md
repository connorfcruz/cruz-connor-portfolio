# Project Overview

## E.M.I. - A Memorization App for Loved Ones

You can find this project [here](Personal-Projects/emi.md).

E.M.I. is a personalized memory-support app designed to help indivudals who have recently experienced dementia or memory loss to reconnect with people, daily routines, and moments in life. By combining familiar faces, images, and gentle reminders, E.M.I. supports memory retention while reducing stress and confusion. The app is built around simplicity and user comfort, and it is accessible for users at various stages of cognitive decline.
